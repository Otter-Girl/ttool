This tool is for converting japanease text to english. It is run out of the command line.

ttool.py takes raw text files and converts them to English
en-hex.py turns the english text into hexidecemil. This module was part of a larger project to translate an old GBA game and is tooled for that task. Some fiddling may be required for desires restult.
cmgben.py is the table of latin charachters to hex codes
